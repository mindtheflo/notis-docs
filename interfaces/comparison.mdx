# Channel Comparison

Notis.ai supports multiple communication channels, each with its own set of features and limitations. This comparison table helps you understand the differences between channels to choose the most appropriate one for your needs.

## Channel-Specific Features

<AccordionGroup>
  <Accordion title="WhatsApp Features">
    - **Quick Reply**: Use WhatsApp's swipe-to-reply feature for contextual responses
    - **Voice Note Transcription**: Automatic transcription of voice messages
    - **Media Sharing**: Share images and documents with context
    - **Location Sharing**: Add location context to notes and tasks
    - **End-to-End Encryption**: All messages are encrypted through WhatsApp's security
  </Accordion>
  
  <Accordion title="Telegram Features">
    - **Bot Commands**: Use slash commands for quick access to features (e.g., `/start`, `/help`)
    - **Inline Mode**: Use Notis in any chat by typing `@NotisAIBot` followed by a command
    - **Rich Message Formatting**: Supports extensive text formatting options
    - **Reactions**: React to messages with emoji
    - **Status Updates**: Receive status updates about message processing
    - **Comments**: List of comments accessible directly from Telegram
    - **2x Boost Mode**: Multiply audio playback speed by two
  </Accordion>
  
  <Accordion title="Email Features">
    - **Email Forwarding**: Forward important emails to save information
    - **Email Templates**: Create templates for recurring types of emails
    - **Rich Text Formatting**: Preserve formatting when saving notes to Notion
    - **Scheduled Emails**: Set up scheduled emails for recurring tasks
    - **Subject Line Commands**: Use email subject line for quick commands
    - **Attachments**: Attach various file types with your emails
  </Accordion>
</AccordionGroup>

## Best Practices for Choosing a Channel

<CardGroup cols={3}>
  <Card
    title="Choose WhatsApp When"
    icon="whatsapp"
  >
    - You need quick, mobile-friendly interactions
    - Voice messages are your preferred input method
    - You want simple, reliable communication with essential features
    - End-to-end encryption is important for your data
  </Card>
  
  <Card
    title="Choose Telegram When"
    icon="telegram"
  >
    - You need advanced formatting and bot features
    - You want to use inline commands in other chats
    - You need features like reactions, comments, and status updates
    - You have longer messages exceeding WhatsApp's character limit
  </Card>
  
  <Card
    title="Choose Email When"
    icon="envelope"
  >
    - You're working from a computer
    - You need to send long-form content
    - You want to use rich text formatting
    - You need to attach multiple files or large documents
    - You want to forward other emails to Notis
  </Card>
</CardGroup>

## Channel-Specific Limitations

<CardGroup cols={3}>
  <Card
    title="WhatsApp Limitations"
    icon="exclamation-circle"
  >
    - **Character Limit**: 1400 characters per message
    - **Voice Message Limit**: 30 minutes maximum
    - **Limited Formatting**: Only basic text formatting
    - **No Slash Commands**: Cannot use command shortcuts
  </Card>
  
  <Card
    title="Telegram Limitations"
    icon="exclamation-circle"
  >
    - **Character Limit**: 4096 characters per message
    - **Bot Restrictions**: Some features limited by Telegram's bot API
    - **Setup Process**: Requires finding and starting the bot
  </Card>
  
  <Card
    title="Email Limitations"
    icon="exclamation-circle"
  >
    - **Response Time**: Slightly slower than messaging apps
    - **No Real-time Status**: Limited delivery and read status
    - **No Voice Messages**: Cannot send voice input directly
  </Card>
</CardGroup>
