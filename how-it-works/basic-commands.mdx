# Basic Commands for Notis.ai

This guide covers the essential commands and instructions you can use with Notis.ai across all supported platforms. Learning these basic commands will help you get the most out of your personal voice assistant.

## General Commands

These commands work across all interfaces (WhatsApp, Telegram, Email):

<CardGroup cols={2}>
  <Card
    title="Take a Note"
    icon="note-sticky"

    **Voice**: "Take a note about [your note content]"  
    **Text**: "Note: [your note content]"
  </Card>
  
  <Card
    title="Set a Reminder"
    icon="bell"

    **Voice**: "Remind me to [task] on [date] at [time]"  
    **Text**: "Reminder: [task] on [date] at [time]"
  </Card>
  
  <Card
    title="Create a Task"
    icon="list-check"
  >
    **Voice**: "Add task: [task description]"  
    **Text**: "Task: [task description]"
  </Card>
  
  <Card
    title="Search Your Notes"
    icon="magnifying-glass"

    **Voice**: "Search for [keyword or phrase]"  
    **Text**: "Search: [keyword or phrase]"
  </Card>

## Command Structure

Most Notis commands follow this general structure:

```
[Command Type]: [Content] [Optional Parameters]
```

For example:
- "Note: Meeting ideas for Project X"
- "Reminder: Call John tomorrow at 3 PM"
- "Task: Finish presentation due Friday priority high"

## Date and Time Formats

Notis understands natural language for dates and times:

<AccordionGroup>
  <Accordion title="Relative Dates">
    - "today"
    - "tomorrow"
    - "next Monday"
    - "in 3 days"
    - "next week"
  </Accordion>
  
  <Accordion title="Specific Dates">
    - "April 15"
    - "April 15, 2025"
    - "15/04/2025" (format depends on your region setting)
  </Accordion>
  
  <Accordion title="Time Expressions">
    - "3 PM"
    - "15:30"
    - "morning" (defaults to 9 AM)
    - "afternoon" (defaults to 2 PM)
    - "evening" (defaults to 7 PM)
  </Accordion>
  
  <Accordion title="Combined Expressions">
    - "tomorrow at 3 PM"
    - "next Monday morning"
    - "April 15 at 15:30"
  </Accordion>
</AccordionGroup>

## Platform-Specific Commands

<Tabs>
  <Tab title="WhatsApp">
    ### WhatsApp-Specific Commands
    
    - **Share Contact**: "Share contact: [name]"
    - **Forward Note**: "Forward note [note ID] to [contact]"
    - **Voice Note**: Hold the microphone button and speak
    - **Quick Reply**: Swipe right on a message to quote and reply
  </Tab>
  
  <Tab title="Telegram">
    ### Telegram-Specific Commands
    
    - **/start**: Initialize or restart the bot
    - **/settings**: Access settings menu
    - **/help**: View help documentation
    - **Inline Mode**: Type @NotisAIBot in any chat to access quick commands
  </Tab>
  
  <Tab title="Email">
    ### Email-Specific Commands
    
    - **Subject Line Commands**: Put commands in the subject line for quick processing
    - **Attachments**: Attach files to include them with your notes
    - **Reply Threading**: Keep conversations in the same thread for context
    - **Email Forwarding**: Forward emails to Notis to save important information
  </Tab>
</Tabs>
