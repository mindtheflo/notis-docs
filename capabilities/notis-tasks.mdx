# Notis Tasks

Notis.ai provides powerful task management capabilities similar to ChatGPT, allowing you to create, track, and complete tasks using natural language. This guide explains how to use Notis Tasks effectively.

## Getting Started with Notis Tasks

<Steps>
  <Step title="Enable Tasks Feature">
    If not already enabled, activate Notis Tasks by sending:
    - "Enable tasks feature" to Notis
  </Step>
  
  <Step title="Create Your First Task">
    Add a task using natural language:
    - "Add task: Review quarterly report by Friday"
    - "Create new task to call client tomorrow at 2 PM"
  </Step>
  
  <Step title="View Your Tasks">
    Check your current tasks:
    - "Show my tasks"
    - "What tasks do I have due today?"
  </Step>
</Steps>

## Key Features

<AccordionGroup>
  <Accordion title="Natural Language Task Creation">
    Create tasks using conversational language just like with ChatGPT:
    
    - "I need to submit the proposal by next Tuesday"
    - "Remember that I have to call Sarah about the project"
    - "Add buying groceries to my to-do list"
    
    Notis will automatically parse these statements and create properly formatted tasks.
  </Accordion>
  
  <Accordion title="Smart Task Organization">
    Notis automatically categorizes and prioritizes tasks based on:
    
    - Due dates and time sensitivity
    - Keywords indicating importance
    - Context from your conversation
    - Previous task patterns
    
    You can also manually specify categories: "Add task: Review report #work priority high"
  </Accordion>
  
  <Accordion title="Context-Aware Task Management">
    Notis remembers the context of your conversations:
    
    - "Follow up on that last email we discussed"
    - "Add a task related to our marketing project"
    - "What was that thing I needed to do for the client?"
    
    The system uses conversation history to provide relevant task suggestions and reminders.
  </Accordion>
</AccordionGroup>

## Best Practices

<AccordionGroup>
  <Accordion title="Be Specific">
    Create clear, actionable tasks with specific details:
    
    - Instead of: "Meeting"
    - Better: "Prepare for client meeting on Thursday at 2 PM"
  </Accordion>
  
  <Accordion title="Use Due Dates Wisely">
    Only add due dates to tasks that truly have deadlines. For other tasks, use priority levels instead to avoid false urgency.
  </Accordion>
  
  <Accordion title="Review Regularly">
    Schedule a daily or weekly review of your tasks to stay organized and reprioritize as needed.
  </Accordion>
  
  <Accordion title="Leverage Context">
    Take advantage of Notis's context awareness by referring to previous conversations when creating related tasks.
  </Accordion>
</AccordionGroup>
